La Section du Mineur



Comment miner



Le programme que j'utilise est xmrig, miner conçu spécialement pour miner les monnaies virtuel basée sur la technologie Cryptonote



Je l'utilise sur toutes les platformes: Windows, Linux, Android, Raspberry et autres

XMRig est un CPU mineur de Monero (XMR) hautes performances, avec prise en charge officielle de Windows. Initialement basé sur cpuminer-multi avec des optimisations / réécritures lourdes et la suppression de nombreux codes hérités, depuis la version 1.0.0, il a été complètement réécrit en C ++.

Le mineur est aussi fonctionnel avec les cartes graphiques NVIDIA et AMD.

Les Liens pour télécharger le mineur sont disponible ici.
